== Description ==
This Plugin adds **colored stripes** width **random colors** and **smooth color transitions** to the background of your website.

**Fully customizable in the Pro Version:**
* Fade duration
* Stripe width
* Orientation
* Colors

= Demo =

You can see the plugin working on [this site](http://www.fabulous.digital).

== Installation ==

1. Unzip the `fabulous-background-colors.zip`
2. Upload the `fabulous-background-colors` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's it.
